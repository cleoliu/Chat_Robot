# Wechat ItChat
ItChat is an open source api for WeChat, a commonly-used Chinese social networking app.

# Requirements
* windows
* python 3.6
* ItChat
```pip install itchat```

# Chat_Robot
> Support for translation and word reply
* wechat_robot.py：run wechat robot server
* translate .py：call translate  api
